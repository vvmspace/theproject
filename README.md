# The Project

The Project is an AI-oriented starter kit for building scalable API 

## Quick Start

- Open `docs/` folder, edit project desciption and starter prompt in `docs/START.md`.
- Optional: Configure the project for your goal by editing `docs/*.md`
- Use starter prompt in your favorite Vibe-code editor.

## Project Map

Most `.md` files are written in a declarative style, with a clear focus on use by AI agents.

- [`AGENTS.md`](AGENTS.md) - agent constitution and rules
